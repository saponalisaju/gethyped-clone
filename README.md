# gethyped-clone

A professional React + Vite project setup with Tailwind CSS and ESLint.

## Setup

1. Install dependencies

   ```bash
   npm install
   ```

2. Run the development server

   ```bash
   npm run dev
   ```

## Recommended workflows

- Start local development: `npm run dev`
- Build production output: `npm run build`
- Preview production build: `npm run preview`
- Check code quality: `npm run lint`
- Fix lint issues automatically: `npm run lint:fix`
- Format codebase: `npm run format`

## Project structure

- `src/` — application source files
- `public/` — static assets
- `package.json` — build and tooling scripts
- `eslint.config.js` — lint rules
- `.editorconfig` — editor formatting conventions
- `.prettierrc` — Prettier formatting rules
